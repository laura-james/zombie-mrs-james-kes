CSS (10 mins)

You then should style the pages as follows:

  1. Make the background colour of the body #a5e6b4
  2. Change all text on the body to be “cursive”
  3. Make all text on the body #004531
  4. Centre all the text
  5. Make the size of the h1 tags 50px and #0F5156
  6. Change font of h1 tag to verdana
  7. Make the size of the h2 tags 30px and #b6895e
  8. Add a 5px solid white border to the images
  9. Add a 10px solid #1fa3ac border to the divs
  10. Make the background of the divs white
  11. All buttons should have a background colour red and a padding of 20px
  12. The background colour of button id "button1" should be blue
  13. The background colour of button id "button1" should be green
